# JSME-bst
【非公式】日本機械学会BibTeXスタイルファイルテンプレート

未完成なのでまだ使わないでください．

# 修正が必要な箇所（自分用）
## 全体

## 英語文献
### @book
* `title`がイタリック体になっている．
* `year`の位置が違う．

### @article
* `journal`がイタリック体になっている．
* `year`の位置が違う．

## 日本語文献
### @inbook
* `title`に『』がついている．
* `pages`が頁になっている．
* `year`の位置が違う．

### @article
* `title`に「」がついている．
* `journal`に『』がついている．
* `year`の位置が違う．
* `volume`が第n巻になっている．
* `number`が第n号になっている．
* `pages`が頁になっている．
